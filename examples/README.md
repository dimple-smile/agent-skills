# Agent Skills Examples

各 skill 的测试示例和使用演示。

## 目录

### [dev-log](./dev-log/)

Dev-log 多语言测试。使用 Docker 容器测试 HTTP 日志收集在 13 种编程语言中的工作情况。

```bash
cd dev-log && ./run-tests.sh
```

支持的语言：JavaScript, TypeScript, Python, Go, Rust, Java, C++, C#, PHP, Ruby, Kotlin, Swift, Dart
